# Interfaces Repo

## Start the app

Remote access only with .env file and variables from that.

host and dialect is set individually.

.env

```.env
PORT=3000
DATABASE=
DB_USER=
DB_PASSWORD=
```

To start the app, run...

```bash
npm run dev
```

the cfg and model folders are in source so babel can compile them correctly.
